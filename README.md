Данный репозиторий посвящен моему решению задачи по предсказанию весограбаритных характеристик товаров. Схема итоговоого подхода отражена на рисунке выше.

## Ноутбуки 
- `eda.ipynb` - EDA предложенного даатсета и построение бейзлайна на табличных признаков.
- `calc_image_embeddings.ipynb` - Получение эмбеддингов картинок.
- `image_embeddings.ipynb` - Построение и тестирование моделей поверх эмбеддингов картинок.
- `text_embeddings.ipynb` - Получение эмбеддингов текстов, построение и тестирование моделей поверх эмбеддингов текстов.
- `fusion.ipynb` - Объединение табличных данных, эмбеддингов картинок и текстов. Построение финального решения, анализ ошибок.
